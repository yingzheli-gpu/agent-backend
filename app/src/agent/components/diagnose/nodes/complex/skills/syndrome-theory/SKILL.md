# 辨证理论要点

## 概述

此技能提供中医辨证的核心理论知识，用于指导复杂病例的证型判断。

## 八纲辨证

八纲是辨证的总纲，用于概括疾病的基本属性：

| 纲领 | 含义 | 辨证要点 |
|------|------|----------|
| **表里** | 病位深浅 | 表证：恶寒发热同见；里证：只热不寒或只寒不热 |
| **寒热** | 疾病性质 | 寒证：面色苍白，畏寒；热证：面红，口渴 |
| **虚实** | 正邪盛衰 | 虚证：久病、体弱；实证：新病、体壮 |
| **阴阳** | 总纲 | 阳证：热、实、表；阴证：寒、虚、里 |

### 寒热真假鉴别

| 情况 | 真寒假热 | 真热假寒 |
|------|----------|----------|
| 本质 | 阳虚寒盛 | 热极似寒 |
| 面色 | 面红如妆（浮游之火） | 面色苍白 |
| 口渴 | 口渴喜热饮，饮不多 | 口渴喜冷饮，大量 |
| 舌象 | 舌淡苔白 | 舌红苔黄 |
| 脉象 | 沉微欲绝 | 沉伏有力 |

### 虚实真假鉴别

| 情况 | 至虚有盛候 | 大实有羸状 |
|------|------------|------------|
| 腹诊 | 腹胀但喜按 | 腹满拒按 |
| 疼痛 | 痛势绵绵，喜温喜按 | 痛势剧烈，拒按 |
| 脉象 | 脉大无力 | 脉实有力但隐伏 |

## 脏腑辨证要点

### 肝

- **生理功能**：主疏泄，藏血
- **病理表现**：
  - 肝郁气滞：胸胁胀满，情志抑郁
  - 肝阳上亢：头痛眩晕，面红目赤
  - 肝血不足：视物模糊，肢体麻木

### 脾

- **生理功能**：主运化，统血
- **病理表现**：
  - 脾气虚：食少腹胀，便溏乏力
  - 脾阳虚：畏寒肢冷，水肿
  - 脾不统血：便血，崩漏

### 心

- **生理功能**：主血脉，藏神
- **病理表现**：
  - 心气虚：心悸气短，自汗
  - 心血虚：心悸失眠，面色淡白
  - 心火亢盛：心烦失眠，口舌生疮

### 肾

- **生理功能**：藏精，主水
- **病理表现**：
  - 肾阳虚：畏寒肢冷，腰膝酸软
  - 肾阴虚：五心烦热，潮热盗汗
  - 肾精不足：生长发育迟缓，早衰

### 肺

- **生理功能**：主气，主宣发肃降
- **病理表现**：
  - 肺气虚：咳嗽无力，气短
  - 肺阴虚：干咳无痰，潮热

## 复合证型识别

### 肝郁脾虚

- **症状组合**：胸胁胀满 + 腹胀便溏 + 食欲不振
- **病机**：肝气横逆犯脾，脾失健运
- **关键鉴别点**：情志因素明显，先有肝郁后有脾虚
- **治法**：疏肝健脾
- **代表方**：逍遥散

### 心肾不交

- **症状组合**：心烦失眠 + 腰膝酸软 + 头晕耳鸣
- **病机**：心火上炎，肾水不济
- **关键鉴别点**：虚烦不寐，五心烦热
- **治法**：交通心肾
- **代表方**：交泰丸

### 气血两虚

- **症状组合**：面色萎黄 + 乏力气短 + 心悸失眠
- **病机**：气虚血亏，心神失养
- **关键鉴别点**：病程较长，渐进性
- **治法**：气血双补
- **代表方**：八珍汤

### 痰湿阻肺

- **症状组合**：咳嗽痰多 + 胸闷气短 + 苔白腻
- **病机**：脾虚生痰，上贮于肺
- **关键鉴别点**：痰量多，色白
- **治法**：健脾化痰
- **代表方**：二陈汤合三子养亲汤

## 使用指南

1. 首先进行八纲辨证，判断疾病的基本属性
2. 然后进行脏腑辨证，确定涉及的脏腑
3. 识别是否为复合证型
4. 注意鉴别真假寒热、真假虚实
5. 综合判断，给出最终证型
