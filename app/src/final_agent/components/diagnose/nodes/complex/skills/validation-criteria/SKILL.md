# 验证标准

## 概述

此技能提供辨证结果的验证标准和流程，用于确保诊断的可靠性。

## 验证维度

### 1. 证据充分性验证

检查诊断是否有足够的证据支持：

- [ ] 症状信息是否全面收集？
- [ ] 舌脉信息是否完整？
- [ ] 是否有遗漏的重要问诊内容？
- [ ] 知识图谱查询结果是否支持？
- [ ] 是否有相似医案佐证？
- [ ] 是否有古籍依据？

### 2. 逻辑一致性验证

检查诊断逻辑是否自洽：

- [ ] 主症是否与证型对应？
- [ ] 兼症是否符合病机演变？
- [ ] 八纲判断是否自洽？
- [ ] 脏腑辨证是否合理？
- [ ] 病因病机分析是否通顺？
- [ ] 是否存在矛盾症状未解释？

### 3. 鉴别诊断验证

检查是否排除了相似证型：

- [ ] 是否考虑了所有相似证型？
- [ ] 是否有"一症多因"的情况未辨清？
- [ ] 寒热真假是否辨明？
- [ ] 虚实真假是否辨明？
- [ ] 是否有需要进一步排除的诊断？

### 4. 治法合理性验证

检查治疗方案是否合理：

- [ ] 治则是否与病机相应？
- [ ] 治法是否针对证型？
- [ ] 是否考虑了禁忌？
- [ ] 方药配伍是否合理？
- [ ] 是否考虑个体因素（年龄、体质等）？

### 5. 安全性验证

检查是否存在安全风险：

- [ ] 是否有需要紧急处理的危险信号？
- [ ] 是否有需要排除的严重疾病？
- [ ] 患者是否适合自我调理？
- [ ] 是否需要建议就医？

## 置信度评估标准

| 置信度 | 标准 | 处理方式 |
|--------|------|----------|
| **0.9+** | 有多个相似案例，理论完全符合，所有验证通过 | 可信度高 |
| **0.7-0.9** | 有部分案例支持，理论基本符合，主要验证通过 | 基本可信 |
| **0.5-0.7** | 案例支持不足，存在部分疑点 | 需人工复核 |
| **<0.5** | 证据不足或存在明显矛盾 | 建议重新评估 |

## 危险信号检查清单

以下情况必须建议立即就医：

### 急症信号
- 高热不退（>39°C）
- 剧烈头痛、胸痛
- 呼吸困难
- 意识障碍
- 大量出血

### 需警惕情况
- 症状持续加重
- 治疗无效
- 出现新的严重症状
- 老年、幼儿、孕妇等特殊人群

## 验证流程

```
开始验证
    │
    ├─ 1. 证据充分性 ─── 不足 ─→ 补充信息
    │       │ 通过
    ├─ 2. 逻辑一致性 ─── 矛盾 ─→ 重新分析
    │       │ 通过
    ├─ 3. 鉴别诊断 ──── 遗漏 ─→ 补充鉴别
    │       │ 通过
    ├─ 4. 治法合理性 ─── 不当 ─→ 调整方案
    │       │ 通过
    ├─ 5. 安全性 ────── 风险 ─→ 建议就医
    │       │ 通过
    └─ 计算置信度 ─→ 输出结果
```

## 不合理情况处理

如果验证不通过：

1. **记录不合理点**：明确说明哪个维度不通过
2. **返回分析**：跳回综合分析步骤重新评估
3. **调整诊断**：根据验证反馈修正诊断
4. **重新验证**：再次进行验证（最多迭代 3 次）
